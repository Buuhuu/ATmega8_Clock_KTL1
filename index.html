ATmega8_Clock_KTL1 by Buuhuu

This is a simple feature based implementation of a clock for the ATmega8 microcontroller.

The task of our hardware-software architecture laboratory was to implement a simple binary clock on the ATmega8 microcontroller. After completing this task in at least a half day we figured out some interesting improvements we want to apply to the microcontroller with this project. The hardware we used can be found at the: http://www.conrad.de/ce/de/product/902766/CONRAD-RETRO-SPIEL-PING-PONG/SHOP_AREA_17618&promotionareaSearchDetail=005

Dependencies

avr-libs, avrdude, avr-gcc, avr-g++

Install

After compiling with avr-gcc flash the generated hex file to the microcontroller memory using the following comand: avrdude -P usb -p m8 -c avrispmkII -U flash:w:
